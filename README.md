# Retail Management System

## Overview
A console-based Retail Management System developed in Python that allows users to browse products, manage accounts, place orders, and maintain shopping history.

## Features
- User login and account creation
- Product categories
- Shopping cart
- Wallet management
- Order placement
- Order history
- JSON-based data storage

## Technologies Used
- Python
- JSON
- Dictionaries
- Functions

## How to Run
1. Install Python 3
2. Run:

```bash
python retail_management_system.py
```
